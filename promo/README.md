# Connect 宣传视频 — 创意 & 调研

## 核心创意

一个人坐在电脑前，打开 Connect，依次输入三个需求：「帮我写一份文案」「探讨产品如何增长」「给我画一张图」。点击 Connect 后，视角穿入屏幕，进入赛博世界。信号沿发光电线分流传递，分别找到莎士比亚、史蒂夫·乔布斯和一位画家。他们各自开始工作——莎士比亚埋头写作，乔布斯发表演讲式的观点输出，画家挥笔作画。完成后，数据沿线路汇聚返回屏幕，最终呈现出文案和图片。

## 场景分解

### 第一幕：用户发起

| # | 场景 | 画面描述 | 风格 |
|---|------|----------|------|
| 1 | 入座 | 一个人坐到电脑前，打开 Connect 界面 | 现实 / 暖色调 |
| 2 | 输入需求 | 依次输入三条任务：写文案、探讨增长、画一张图 | 屏幕录制 + UI 动画 |
| 3 | 点击 Connect | 按下按钮，屏幕泛起光芒，视角被吸入屏幕 | 现实→赛博过渡 |

### 第二幕：信号流转

> **视觉重点**：这一幕是全片最华丽的部分，信号在光影世界中拐弯抹角地穿梭，最终分屏呈现三个目标。

| # | 场景 | 画面描述 | 风格 |
|---|------|----------|------|
| 4 | 进入赛博世界 | 穿过屏幕，进入发光电线交织的数字空间 | 赛博朋克 / 霓虹蓝紫 |
| 5 | 单线追踪 | 跟随一根主信号线，在光影世界中高速穿梭——转弯、穿隧道、掠过数据节点，像过山车一样沉浸式推进 | 赛博 / 第一人称视角 / 速度感 |
| 6 | 信号分流 | 主线到达一个巨大的发光枢纽节点，一分为三，三条光线各自射向不同方向 | 赛博 / 粒子爆发特效 |
| 7 | 第一条线到达 | 镜头跟随第一条信号线继续穿梭，穿过时空隧道般的光廊，最终抵达莎士比亚的书房。信号化为一个光点落在他的书桌上 | 赛博→古典欧洲过渡 / 烛光暖调 |
| 8 | 三分屏展开 | 画面切为三分屏，同时展示三条信号线的终点：(左) 莎士比亚已收到光点 (中) 乔布斯的极简办公室亮起光点 (右) 画家的画室亮起光点 | 三分屏 / 各自风格 |

### 第三幕：协同办公

> **视觉重点**：保持三分屏构图，三个人物同时工作，各有特色。

| # | 场景 | 画面描述 | 风格 |
|---|------|----------|------|
| 9a | 莎士比亚写作 | （左屏）收到光点后，拿起羽毛笔蘸墨，埋头在羊皮纸上奋笔疾书 | 古典 / 烛光特写 |
| 9b | 乔布斯输出 | （中屏）收到光点后，面对屏幕开始说话输出观点（无声，嘴在动，配合字幕或文字浮现） | 现代极简 / 苹果风 |
| 9c | 画家作画 | （右屏）收到光点后，调色、挥笔，画布上逐渐浮现作品 | 艺术感 / 延时摄影 |
| 10 | 完成信号 | 三个分屏中，人物各自完成任务，各自亮起完成的光点 | 三分屏 / 光点亮起 |

### 辅助工具

| 工具 | 用途 |
|------|------|
| Midjourney / FLUX | 生成关键帧图片，再用图生视频 |
| 剪映专业版 | 剪辑拼接 + 配乐 + 字幕 |
| DaVinci Resolve | 专业调色 + 剪辑（免费） |
| Remotion | 用 React 代码做 UI 动画（与项目技术栈一致） |

## 推荐制作流程

1. **关键帧设计** — 用 Midjourney/FLUX 为每个场景生成静态概念图
2. **图生视频** — 用 Kling/Vidu/Runway 将关键帧转为动态片段
3. **UI 动画** — 用 Remotion 或屏幕录制做 Connect 界面操作部分
4. **剪辑合成** — 用剪映/DaVinci 拼接所有片段，加转场、配乐、字幕
