# Connect 宣传视频 — 制作流程

> 工具链：即梦（Seedance 2.0 生图+生视频） → 可灵（备选） → 剪映（剪辑合成）
> 预计镜头数：19 + 1 结尾 = 20 个片段

---

## 阶段一：角色设定（先做，后续所有镜头 @引用）

在即梦中用生图功能，生成所有角色的参考图，后续每个镜头都 @这些图来锁定角色一致性。

> **生图比例说明**：
> - 角色参考图（肖像/半身）→ **1:1**（正方形，适合人物特征锁定）
> - 角色参考图（全身/动作姿态）→ **3:4**（竖版，展示完整身体和服装）
> - 场景关键帧（阶段二）→ **16:9**（横版，匹配视频输出比例）

## 阶段三：关键帧 → 视频生成

在即梦中，用阶段二的关键帧图片 + 角色参考图 → 生成视频片段。

每个镜头的输入格式：

```
@主角-背影 @主角-手部特写
@[阶段二生成的关键帧图片]

[阶段二的提示词]，镜头缓慢推进，电影感运镜，带环境音效

--时长 5秒
--分辨率 1080p
--音频 开启
```

### 生成顺序建议

| 优先级 | 镜头 | 原因 |
|--------|------|------|
| 先做 | Shot 4/5/6 赛博穿梭 | 全片视觉核心，可能需要多次迭代 |
| 再做 | Shot 9a/9b/9c 三人工作 | 角色镜头，需要确认一致性 |
| 然后 | Shot 1/3/12/13 现实场景 | 相对简单 |
| 再然后 | Shot 7/8/10/11 过渡镜头 | 依赖前面镜头的风格定调 |
| 最后 | Shot 14-19 喜剧段 | 节奏依赖剪辑，生成压力小 |

### 不满意时切可灵

如果即梦生成的某个镜头效果不好，用同样的关键帧图片导入可灵 Kling 重新生成：
- 赛博长镜头（Shot 5）→ 可灵支持更长时长，更适合连续运动
- 人物特写（Shot 9a/9b/9c）→ 可灵的人脸一致性也很强

## 阶段五：剪映合成

### 输入素材清单

```
/素材/
├── 角色参考图/        ← 阶段一的 18 张
├── 关键帧/            ← 阶段二的每镜头 1 张
├── AI视频片段/        ← 阶段三的 ~16 个视频文件
├── UI录制/            ← 阶段四的 3 个屏幕录制
└── 音乐/              ← 背景音乐（见下方）
```

### 剪映时间线操作

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 导入所有片段 | 按 Shot 编号排列 |
| 2 | 粗剪拼接 | 按剧本顺序排列，确认叙事流畅 |
| 3 | 分屏处理 | Shot 8/9/10 → 使用剪映"分屏"模板，左中右三等分 |
| 4 | 转场 | Shot 3（现实→赛博）和 Shot 12（赛博→现实）加光效转场 |
| 5 | 字幕 | Shot 9b 乔布斯的浮动文字、Shot 19 分身对话字幕 |
| 6 | 音效修正 | AI 生成的音效不完美的地方，用剪映音效库替换/叠加 |
| 7 | 背景音乐 | 第一到四幕：电子/史诗配乐；第五六幕：切换为轻快喜剧配乐 |
| 8 | 结尾 Logo | 黑底 + Logo 图片 + 文字动画 + 钢琴单音 |
| 9 | 导出 | 1080p / 30fps / H.264 |
