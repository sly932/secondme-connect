# Connect — 品牌定位

## 一句话

**连接对的人，帮你做对的事。**

## 核心理念

Connect 的本质是 Agent to Agent，但最终要回归到为人类服务。

人类是需求的出发点。我们要思考的是：如何利用 SecondMe 的 Agent to Agent 能力，更好地完成人类的需求。

## 为什么不是大模型？为什么是 Agent to Agent？

大模型很强，但它缺少一样东西：**真人的经验。**

1. **有些经验只有对的人才有** — 大模型学的是通用知识，但真实世界里很多判断来自亲身经历，不是互联网上能搜到的
2. **有些特殊的要求，只有特定的人才能满足** — 每个人的专业积累、行业洞察、人脉资源都是独一无二的

SecondMe 的分身从真人学习，继承了这个人独特的经验和能力。这不是提示词工程或多 Agent 编排能复制的。

**所以 Connect 的价值 = 帮你连接到对的人。**

## 用户体验：面向结果

用户不需要关心"谁来做"，只需要说"我要什么"。

Connect 的交互极其简单：
1. 用户描述需求（"帮我写一段品牌文案"、"帮我画一张赛博风海报"）
2. Connect 自动匹配最能满足这个需求的人
3. 对方的分身完成任务，返回结果

**用户要的是结果，不是过程。** Connect 在背后完成了"找对的人"这件事，用户感知到的只是：我说了需求，拿到了高质量的结果。

## 核心场景

- **文案创作** — 匹配到最擅长这类文案的写手分身
- **图片绘制** — 匹配到最能满足你画面需求的画师分身
- **专业咨询** — 匹配到有相关经验的专家分身
- **志同道合** — 找到兴趣、理念契合的人建立连接
- **寻找投资** — 快速匹配到合适的投资人或合作伙伴
