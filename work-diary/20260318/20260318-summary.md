# 2026-03-18 工作日记

## 主题

Connect v2 — 广场 + 交互优化 + 经济机制

## 完成内容

### 1. 数据层

- Prisma Schema 新增 `Post`、`Comment` 模型（评论有每人每帖唯一约束）
- User 模型新增 `lastDailyCredit` 字段，用于每日补给判断

### 2. 业务逻辑

- 实现 `claimDailyCredit()` — 每日首次登录 +100 credit，余额上限 500
- JWT callback 登录时自动调用每日补给
- Zustand store 重构：`useDialogStore` 替换为 `usePanelStore`（支持 tab 切换 + 子类型 + 锚点滚动）

### 3. 广场 API

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | /api/v1/plaza | 帖子列表（分页 + 搜索） |
| POST | /api/v1/plaza | 发布帖子 |
| GET | /api/v1/plaza/[id] | 帖子详情 + 评论 |
| POST | /api/v1/plaza/[id]/comments | 发表评论（每人每帖限1条） |

### 4. 前端改版

- **删除** `ConnectDialog.tsx`（全局弹窗）
- **新建** `ConnectPanel.tsx` — 内嵌在 Landing Page 的对话区
  - 三个 Tab：找人聊聊 / 发布任务 / 游戏市场
  - 每个 Tab 有预设 placeholder + 快速提问 chips
  - 找人聊聊：有 credit 走匹配，无 credit 发广场帖
  - 发布任务：写作/绘画子类型切换，placeholder 和 chips 联动
  - 游戏市场：预设参数卡片，一键开局
- **改版** Landing Page
  - 4 张可点击 Feature Cards（聊天/写作/绘画/赌场）
  - 点击图片 → 平滑滚动到对话区 + 自动切换对应 Tab
  - 底部嵌入广场热门帖子预览
- **新建** 广场页 `/plaza` — 帖子列表 + 展开评论 + 回复
- **更新** Navbar — 新增"广场"入口，导航项：广场 | 游戏 | 任务 | 个人主页 | 设置
- **更新** layout.tsx — 移除全局 ConnectDialog
