# 新增模拟真实用户 NPC 档案

**目标**：补充一批模拟真实普通人的 NPC，让广场更有真实感，体现"连接真实的人"的价值。

## 背景

现有 65 个 NPC 全是名人，评委反馈：名人分身大模型本身就能模拟，没有差异化。
真正的价值在于连接**真实的人的独特技能和经验**。

## 行业分类（约 50-55 人）

### 1. 开发者（10-12 人）⭐ 核心类别
- 前端开发（React/Vue/小程序）
- 后端开发（Java/Go/Python）
- 独立开发者（出海工具/SaaS）
- AI 应用开发（RAG/Agent/prompt engineering）
- 全栈 / Web3 开发
- 移动端开发（iOS/Android/Flutter）
- 运维/DevOps/云原生
- 数据工程/大数据

### 2. 产品经理（8-10 人）⭐ 核心类别
- ToC 产品（社交/电商/内容）
- ToB 产品（SaaS/企业服务）
- AI 产品（大模型应用/Agent）
- 数据产品
- 电商产品
- 增长产品
- 硬件/IoT 产品
- 海外产品

### 3. 创业者（8-10 人）⭐ 核心类别
- 项目和过往经历要**丰富、真实**
- 涵盖不同阶段：种子轮/天使轮/A 轮/B 轮+
- 涵盖不同赛道：AI、消费、SaaS、硬件、教育、医疗、出海等
- 有成功的也有正在探索的

### 4. 投资人（6-8 人）⭐ 核心类别
- 需要有**具体的投资经历**（投过哪些品牌/项目）
- 记录**投资偏好**：
  - 偏好什么样的创始人（背景、性格）
  - 偏好什么样的项目（阶段、模式）
  - 偏好什么行业/赛道
- 类型覆盖：天使投资人、早期 VC、成长期 VC、产业基金、FA、个人 LP

### 5. 设计类（3 人）
- UI/UX 设计师
- 品牌/平面设计师
- 3D / 动效设计师

### 6. 商业运营类（3-4 人）
- 跨境电商卖家（亚马逊/TikTok Shop）
- 增长运营（私域/社群/投放）
- 市场营销 / SEO

### 7. 专业服务类（3-4 人）
- 律师（知识产权/合同/劳动法）
- 财务/税务顾问
- HR / 猎头
- 心理咨询师

### 8. 内容创作类（3-4 人）
- 网络小说家（连载/网文）
- 公众号/自媒体写手
- 短视频编导/文案
- 技术博主 / 知识分享者

## 档案要求

每个人物需要包含：
- **bio**：完整的背景/性格/价值观/MBTI 分析
- **shades**：兴趣标签（职业技能 + 个人爱好如健身、摄影、旅行、游戏、咖啡、读书等）
- **softmemory**：知识库（具体的经历、事实）
- **autoMemory**：可预填一些模拟数据
